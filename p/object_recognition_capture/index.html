README

Capture

A toolset for capturing 3D models of objects from set of views.

get it

% git clone git://github.com/wg-perception/capture.git

requires

  • boost
  • cmake
  • OpenCV > 2.3 http://opencv.willowgarage.com/wiki/
  • ecto https://github.com/plasmodic/ecto

build it

% cd capture
% mkdir build
% cd build
% cmake ..
% make

    README

    Capture

    A toolset for capturing 3D models of objects from set of views.

    get it

    % git clone git://github.com/wg-perception/capture.git

    requires

    • boost
    • cmake
    • OpenCV > 2.3 http://opencv.willowgarage.com/wiki/
    • ecto https://github.com/plasmodic/ecto

    build it

    % cd capture
% mkdir build
% cd build
% cmake ..
% make
