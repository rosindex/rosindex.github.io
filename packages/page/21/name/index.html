1 2011-05-27 eigen
This package contains version 3 of the Eigen C++ template library for linear algebra. It automatically downloads, extracts a tarball. For each major release of the Eigen library, the tarball will get updated. This ROS package does not intend to modify Eigen in any way, it simply provides a convenient way to download and compile the Eigen library. When Eigen becomes available from the major OS package managers, this ROS package is likely to be replaced by a system dependency. This package contains Eigen3 with EIGEN2_SUPPORT enabled. In future versions EIGEN2_SUPPORT will be removed.
