1 2015-01-11 actuator_array_gazebo_plugin
This package provides a Gazebo Plugin to control an array of position-controlled actuators, such as RC Servos. In addition to the standard parameters for update rate and robot namespace, a series of "joint" elements are provided. Each "joint" element must provide a "name" and, optionally, a "home" position and a set of PID parameters that control the joint stiffness inside Gazebo. Joint limits, max velocities, etc. are read from the "robot_description" parameter.

1 2011-03-23 wbc_pr2_ctrl
This package contains an adaptation of the Stanford Whole-Body Control Framework (WBC) for PR2. It relies mainly on the joint-space dynamic model which is part of WBC. There is a "pump" controller plugin which sends robot state from real-time to user-space and the torque commands back from user-space into the real-time controller. For simpler WBC operations, it would (probably) be possible to run the entire WBC servo within the plugin, but tests in late 2009 (which were based on a very complete set of WBC functionality) failed in this respect because they were too slow to fit within the 1kHz RT control loop on PR2.
