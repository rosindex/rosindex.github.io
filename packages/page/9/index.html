Parameters it uses: display - shows the image on the local computer cameraguid - hex string, if specified will only open a camera with this guid cameraindex - index of the camera to start if cameraguid is not present mode - any one of opencv modes to set, for example: "MODE_640x480_YUV422" framerate - number of frames a second to publish KK_fx,KK_fy,KK_cx,KK_cy - intrinsic parameters such that KK = [fx 0 cx; 0 fy cy; 0 0 1] kc_k1,kc_k2,kc_p1,kc_2 - radial distortion coefficients to undistort the image colorfilter - color conversion for debayering, can be "COLOR_FILTER_RGGB", "COLOR_FILTER_GBRG", "COLOR_FILTER_GRBG", "COLOR_FILTER_BGGR" brightness contrast whitebalance_blueu whitebalance_redv exposure compression - how to compress the data before sending, can be "none", "png", "jpg", "jpeg" gain - the gain on the camera values
