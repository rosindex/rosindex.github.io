1 2015-01-30 cob_hardware_interface
This package implements the cob_hardware_interface. The interface uses topic communication rather than direct pointer access to the drivers. I.e. it publishs the desired velocity to a topic and reads from the /joint_states topic. Thus, it can be used for both velocity-controlled real hardware as well as velocity-controlled simulation (i.e. using the JointVelocityControllers)
