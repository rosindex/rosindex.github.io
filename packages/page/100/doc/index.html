This package contains a variety of nodes that distribute requests for detection and for collecting the responses from detectors. The code for these nodes is found in src/batchdetector/nodes. Collator collects hypotheses generated by a set of detectors, and outputs images with bounding boxes from this collection. This node is geared towards producing results for the SRVC. Dispatcher listens for new scenes that are ready for processing and sends messages that dictate which detectors are run on the scenes. Currently it simply sends out messages to all detectors.
