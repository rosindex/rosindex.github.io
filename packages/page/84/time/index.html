1 2013-11-13 scene_server
Scene server is a data storage node that allows numerous visual processing algorithms such as object recognizers and 3D spatial reasoning nodes to run on the same set of data. The function of scene server is to synchronize incoming sensor information, serve this information to various processing nodes, and finally to correlate the results of each of these nodes in a consistent format to be viewed by the remainder of the system.
